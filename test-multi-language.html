Backend I18nManager

// Core translation method i18n.t('app.title') // Returns: "MIC Browser Ultimate - The Future of Web Automation" i18n.t('navigation.back') // Returns: "Back" (or translated equivalent) // Language switching await i18n.setLanguage('es') // Switches to Spanish await i18n.setLanguage('ja') // Switches to Japanese // Get available languages const languages = i18n.getAvailableLanguages() // Returns: [{code: 'en', name: 'English', nativeName: 'English'}, ...]

Client-Side Usage

<!-- HTML with translation attributes --> <button data-i18n="navigation.back">Back</button> <input data-i18n-placeholder="navigation.addressBar" /> <span data-i18n="app.title">MIC Browser Ultimate</span> // JavaScript translation window.i18n.t('settings.language.interfaceLanguage') // "Interface Language" await window.i18n.changeLanguage('fr') // Switch to French
